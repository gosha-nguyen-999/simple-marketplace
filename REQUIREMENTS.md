# Game Asset Marketplace — Requirements

## What It Is
A marketplace website where players can browse and list in-game assets for sale across multiple games.

## V1 Scope
Frontend showcase only — browse listings, no real auth/checkout needed.

## Supported Games
- CS2 (skins, knives, gloves)
- Valorant (skins, agents, sprays)
- Fortnite (skins, pickaxes, emotes)
- Roblox (items, accessories, game passes)
- + "Other Games" catch-all category

## Visual Style
Bold & colorful — vibrant, energetic, game-inspired. Dark background with neon/colorful accents.

## Pages / Sections
1. **Hero** — bold headline, search bar, featured game filters
2. **Featured Listings** — grid of asset cards with price, game tag, seller
3. **Browse by Game** — game category pills/tabs to filter listings
4. **Asset Card** — image, name, game, price, condition (Factory New etc.), seller name, "Buy Now" button
5. **Footer** — links, tagline

## Asset Categories per Game
- CS2: Skins, Knives, Gloves, Cases
- Valorant: Gun Skins, Agents, Bundles
- Fortnite: Outfits, Pickaxes, Emotes, V-Bucks
- Roblox: Accessories, Game Passes, Limiteds

## Sample Listings (mock data)
12-16 realistic asset listings with names, prices, game tags, rarity labels

## Tech
Next.js + Tailwind, same stack as quiz-project
